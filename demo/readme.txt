Demonstration of SysRes DPOAE data analysis

can4.mat - DPOAE measurement from a human ear
	This file is in MATLAB format.
	The data were recorded using the SysRes program.

can4.lst - SysRes script to view the above data file.
	Type "sysres can4" to execute this list of commands.
	A spectrum of the recorded waveform is displayed.
	Press any key to continue.
	A temporal envelope of the DP is displayed.
	The envelope is computed using heterodyne analysis.
	A two-exponential function is fit to the envelope.
	Press any key to continue and "q" to quit.

u8l19b.mat - DPOAE measurement from a mouse ear
	This file is in MATLAB format.
	The data were recorded using the SysRes program.

dpadem.lst - SysRes script to view the above data file.
	Type "sysres dpadem" to execute this list of commands.
	Two temporal envelopes of the DP are superimposed to
	compare the heterodyne and short-term FFT analysis methods.
	Press any key to continue.
	A two-exponential function is fit to the heterodyne envelope.
	Press any key to continue.
	A two-exponential function is fit to the magnitude and phase
	of the heterodyne analytic signal.
	Press any key to continue and "q" to quit.
